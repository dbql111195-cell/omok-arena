오목대전 테스트 방법

1. 혼자 테스트
- index.html을 열고 닉네임 입력 후 입장합니다.
- 연습 상대와 대국을 누르면 바로 둘 수 있습니다.

2. 방 코드 테스트
- 첫 번째 화면에서 입장 후 방 만들기를 누릅니다.
- 방 코드나 초대 링크를 복사합니다.
- 두 번째 화면에서 같은 링크를 열거나 방 코드를 입력하고 입장을 누릅니다.

3. 친구와 실제로 두기
- file:/// 주소는 내 컴퓨터 파일이라 친구가 접속할 수 없습니다.
- 친구와 하려면 이 폴더를 서버로 실행하거나 웹에 배포해야 합니다.
- 서버 실행 예시:
  node server.js
- 실행 후 같은 와이파이에서는 내 컴퓨터 IP와 포트로 친구가 접속할 수 있습니다.
- 인터넷 친구와 하려면 배포 주소가 필요합니다.

4. 배포 주소 만들기
- 이 폴더 전체를 GitHub 저장소에 올립니다.
- Render, Railway, Fly.io 같은 Node 서버 배포 서비스를 연결합니다.
- Start command는 npm start 입니다.
- 배포가 끝나면 https://... 주소가 생깁니다.
- 그 주소에서 방 만들기를 누르고 초대 링크를 친구에게 보내면 됩니다.
